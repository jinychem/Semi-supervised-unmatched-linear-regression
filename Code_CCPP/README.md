Codes:

1. main code
"ccpp.R": computes SSLEMLEs and OLSEs

2. auxiliary codes
"noise_distribution.R": generates noise variables (Gaussian/Laplace)
"loglikelihood.R": computes log-likelihood of matched and unmatched parts

3. RDS files
SSLEMLEs, OLSEs and matched MLEs from the main code.

4. Combined Cycle Power Plant data set
Folds5x2_pp.xlsx: publicly available from the UCI Machine Learning Repository at https://doi.org/10.24432/C5002N
